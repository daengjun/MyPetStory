# MyPetStory
펫들의 일상을 기록하는 SNS소셜 앱

# 화면 구성
1. 메인 타임 라인
2. 서브 타임 라인
3. 새 게시글 작성
4. 내 정보 & 펫 목록
5. 친구 목록 & 채팅방
